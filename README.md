
# To-Do List App

Aplicación web de lista de tareas construida con:

- HTML5
- CSS3 (BEM)
- JavaScript (DOM)
- localStorage

## Funcionalidades

- ✅ Agregar tareas
- ✅ Eliminar tareas
- ✅ Marcar como completadas
- ✅ Persistencia con localStorage

## Objetivo

Proyecto desarrollado para practicar manipulación del DOM y gestión de estado en frontend.
